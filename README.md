# chat-room局域网聊天室

基于Qt语言的局域网聊天室项目，可以实现基本的用户注册，登录，找回密码，群聊，私聊，统计当前在线人数以及传送文件等等功能
此外，在github中常见版本的基础上增加了一些功能如截图保存，实时语音，侧边栏自定义等
P.S.其中截图的保存地址采用系统绝对路径，需要自行修改！
